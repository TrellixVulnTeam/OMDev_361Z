** Version Française en fin de fichier **

First use of VisualFigaro : 

  - Launch the editor Jedit from the Start menu,
  - In Jedit, launch the VisualFigaro plugin via the plugin menu,
      and choose the option "dock left"

When installing the VisualFigaro plugin for the first time on a PC,
you may encounter intialization problems. You can solve them by 
suppressing all the contents of the directory called .Jedit in 
Documents and settings\user name



Première utilisation de VisualFigaro :

  - Lancer l'éditeur Jedit depuis le menu démarrer,
  - Dans Jedit, lancer le plugin VisualFigaro par le menu plugin,
       et choisir l'option "dock left"

Pour l'installation sur un nouveau poste, il peut y avoir des problèmes d'initialisation.
On peut les résoudre en supprimant tout le contenu du répertoire de nom .Jedit dans 
Documents and settings\nom utilisateur
